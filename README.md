## Sääntelytulkki ⚡

Tekoälypohjainen työkalu energia-alan säädösten tulkintaan selkokielellä. Rakennettu Python, Streamlit ja Claude API:lla.

## Ominaisuudet
- Kysy energia-alan sääntelystä luonnollisella kielellä
- Vastaukset viittaavat aina alkuperäiseen säädökseen ja pykälään
- Tietokanta kattaa sähkömarkkinalain, energiatehokkuuslain ja EU-direktiivit
- Suunniteltu energia-alan ammattilaisten käyttöön

## Teknologiat
Python
Streamlit
Claude API (Anthropic)
